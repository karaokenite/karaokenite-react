## Local Development

First, install everything once using [yarn](https://yarnpkg.com/):

```shell
yarn
```

Now, in two separate terminals, you can start the server with `yarn dev:server` and the site with `yarn dev:server`:

```shell
yarn dev:server
```

```shell
yarn dev:site
```
