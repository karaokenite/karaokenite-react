## Local Development

First, install everything once using [yarn](https://yarnpkg.com/):

```shell
yarn
```

Now, you'll be able to start two terminals:

1. Running the backing Node server:

   ```shell
   yarn dev
   ```

2. Running the Next.js client serer:

   ```shell
   yarn serve
   ```
